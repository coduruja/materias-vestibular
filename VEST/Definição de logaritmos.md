## Sobre o que se trata.

Define logaritmo como o expoente necessário para obter um número:

$log_b(x) = y ↔ b^y = x$

Inclui propriedades fundamentais:

- $log(ab) = log a + log b$
- $log(a/b) = log a − log b$
- $log(a^n) = n · log a$
